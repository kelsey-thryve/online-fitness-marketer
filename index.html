FAQ

Common questions

What's a "launch"?
Each launch is one full marketing kit tailored to a specific challenge or program you're running — e.g. a 6-Week Fat Loss Challenge. You get all 7 deliverables built around that one launch.
How long does generation take?
3-5 minutes once the form is submitted. The seven assets are generated sequentially so each one builds on the previous outputs.
Is this just AI templates?
No. The prompts are tailored using the answers you provide — different challenges, audiences and voices generate materially different launches.
Can I edit the assets?
They're plain-text documents. Copy them, paste them into your tool of choice, edit anything. You own the output.
What if I run more than 3 challenges?
The 3-launch limit covers most solo coaches. Need more? Reach out and we'll set you up with a higher tier.
